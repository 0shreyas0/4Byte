# Agent Skills

These skills are reusable checklists and execution prompts for agents working in this repository.
Use them after reading the mandatory instruction files in `.agents/instructions/`.

## Available skills
- `code-check.md`: broad code review and implementation hygiene sweep.
- `folder-structure-scan.md`: verifies feature placement and project organization.
- `theme-conventions-check.md`: audits semantic colors, typography roles, and mode compatibility.
- `documentation-sync.md`: ensures new surfaced features are registered in docs and previews.

## Usage pattern
- Pick one or more skills based on the task.
- Treat each skill as an operational checklist, not just a suggestion.
- Report findings with file references and concrete fixes when possible.
